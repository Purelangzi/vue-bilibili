# vue-bilibili

仿b站移动端开发，使用vue-cli4.x脚手架创建项目

技术栈 vue2.6 + vue-router + axios + ES6 + less + flex + vant

vue路由采用history模式和懒加载的方式

使用vw单位实现自适应布局

使用阿里妈妈矢量图标库（iconfont）
